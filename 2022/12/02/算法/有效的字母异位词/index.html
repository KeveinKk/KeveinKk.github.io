有效的字母异位词

题目

给定两个字符串s和t，编写一个函数来判断t是否是s的字母异位词。（若s和t中每个字符出现的次数都相同，则称s和t互为字母异位词）

方法一

分析

这道题其实和昨天的题（字符串中第一个唯一字符）非常类似，那么我很自然的就想到用一个times[26]数组统计每个字符串中的每个字符出现的次数，然后比较每个字符串的times[26]数组，如果两个字符串的这个两个数组相同，那么它们就是字母异位词，反之就不是。

  • 时间复杂度：只用将每个字符串遍历一遍就可以得到它的times[26]数组，然后比较两个times[26]数组是通过调用Arrays.equals()方法，它在底层也是将两个数组再遍历一遍，因此时间复杂度是O(N)O(N)O(N)
  • 空间复杂度：O(N)O(N)O(N)，申请了两个times[26]数组

基础知识

  • Arrays.equals(int[] a, int[] a2)

    Returns true if the two specified arrays of ints are equal to one another. Two arrays are considered equal if both arrays contain the same number of elements（两个数组包含的元素个数相同）, and all corresponding pairs of elements in the two arrays are equal（两个数组中对应的元素相同）. In other words, two arrays are equal if they contain the same elements in the same order（长度相同、元素顺序相同）. Also, two array references are considered equal if both are null（两个数组都为空时也相同）.

    • 在Arrays类中 equals的实现是这样的
    • ArraysSupport.mismatch(a,a2,length)

    The mismatch() is a method that is defined under the Arrays class of Java.util package and it is used with respect to the two arrays passed as an argument in the mismatch method. This method returns the index at which two arrays passed as a parameter to the mismatch() function have the first unequal element（返回第一个不相等元素的下标）. It is quite useful to check whether two arrays contain the same corresponding elements or not（用于判断两个数组是否拥有对应相同的元素）. This responds when a mismatch occurs. 方法二

分析

还有一种方法也可以快速的判断两个字符串中字符出现的频率是否是相同的。t 是s的异位词等价于两个字符串排序后相等。因此我们可以对字符串s和t分别排序，看排序后的字符串是否相等即可判断。

  • 时间复杂度：通过调用Arrats.sort()方法来对字符串进行排序，时间复杂度为O(NlogN)O(NlogN)O(NlogN)，然后再判断两个排好序的数组是否相等，时间复杂度为O(N)O(N)O(N)，总的时间复杂度为O(NlogN)O(NlogN)O(NlogN)
  • 空间复杂度：不需要额外的存储空间，O(1)O(1)O(1)
