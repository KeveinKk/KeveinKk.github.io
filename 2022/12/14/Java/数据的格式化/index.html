数据的格式化

日期格式化

Date

在java.util包中存在一个Data类，提供对日期的描述，这个描述可以精确到毫秒

The class Date represents a specific instant in time, with millisecond precision.

public Date()

Allocates a Date object and initializes it so that it represents the time at which it was allocated, measured to the nearest millisecond.

  • 获取系统当前时间（精确到毫秒）
  • 直接通过无参的构造方法获得

public Date(long date)

Allocates a Date object and initializes it to represent the specified number of milliseconds since the standard base time known as “the epoch”, namely January 1, 1970, 00:00:00 GMT.

  • 通过传入一个自1970年1月1日00:00:00 000开始的毫秒数来构造一个Date对象

  • 之所以是八点，是因为时区的问题，这里显示的是北京时间（东八区）

SimpleDateFormat

在java.text中存在SimpleDateFormat类，提供用特定格式对日期的描述

SimpleDateFormat is a concrete class for formatting and parsing dates in a locale-sensitive manner. It allows for formatting (date → text), parsing (text → date), and normalization.

public SimpleDateFormat(String pattern)

Constructs a SimpleDateFormat using the given pattern and the default date format symbols for the default FOMAT locale. Note: This constructor may not support all locales.

  • 用给定格式创建一个SimpleDateFormat对象
  • 格式

public final String format(Date date)
  • 将一个Date类型的时间转换成一个字符串

public Date parse(String source) throws ParseException
  • 将给定的一个时间字符串转换成标准的Date类型时间
  • 注意SimpleDateFormat的日期格式必须和字符串的日期格式相同

时间

public static native long currentTimeMillis()

Returns the current time in milliseconds. the difference, measured in milliseconds, between the current time and midnight, January 1, 1970 UTC

  • 返回自1970年1月1日00:00:00 000到当前系统时间的总毫秒数

程序运行时间的计算

  • 通过分别获取某段程序运行前的系统时间，以及程序运行结束后的系统时间，通过做差来求出该程序的运行时间

获取昨天当前的时间

数据格式化

DecimalFormat

在java.text包下存在DecimalFormat包，提供对数字的描述

It has a variety of features designed to make it possible to parse and format numbers in any locale, including support for Western, Arabic, and Indic digits. It also supports different kinds of numbers, including integers (123), fixed-point numbers (123.4), scientific notation (1.23E4), percentages (12%), and currency amounts ($123). All of these can be localized.

public DecimalFormat(String pattern)

Creates a DecimalFormat using the given pattern and the symbols for the default FORMAT locale. This is a convenient way to obtain a DecimalFormat when internationalization is not the main concern.

  • 用给定的模式去创造一个DecimalFormat对象

  • 简单来说
    • *#*代表任意数字
    • 0代表不够时的补位
    • ，代表千分位
    • *.*代表小数点

public final String format(double number)

  • 将一个double类型的数据转换成特定的DecimalFormat类型的字符串

BigDecimal

The BigDecimal class provides operations for arithmetic, scale manipulation, rounding, comparison, hashing, and format conversion. The toString() method provides a canonical representation of a BigDecimal.

  • 精度极高的大数据，不属于基本数据类型，一般用于财务软件中
  • 使用BigDecimal中提供的方法来进行数据操作

随机数

在java.util包下有Random类，用于生成随机数

public int nextInt(int bound)

Returns a pseudorandom, uniformly distributed int value between 0 (inclusive) and the specified value (exclusive), drawn from this random number generator’s sequence.

  • 返回一个在$[0,bound)$之间的随机整数

public int nextInt()

  • 返回一个整形范围内的随机数

Math.Random()

还可以用Math类中的Random()方法来生成$[0,1)$之间的随机浮点数

  • 用Math.Random()生成$[0,a)$之间的随机数方法

枚举

  • 枚举是一种特殊的类，在编译之后同样生成一个.class文件
  • 枚举也是一种引用类型
  • 枚举中的每一个值可以看作是一个常量，我们不需要再为其赋值

枚举的语法格式

  • 枚举值的访问
    • 通过枚举类型名去访问
  • 枚举值的命名全部采用大写字母

枚举的使用

  • 枚举的使用有点类似于布尔类型
