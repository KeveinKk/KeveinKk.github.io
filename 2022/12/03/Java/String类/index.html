String类

String的存储原理

The String class represents character strings. All string literals in Java programs, such as "abc", are implemented as instances of this class.

Strings are constant; their values cannot be changed after they are created（String是常量，创建之后无法修改）. String buffers support mutable strings. Because String objects are immutable they can be shared.

  • String表示字符串类型，属于引用数据类型

    • 在Java中所有用双引号修饰的都是String对象

  • 在Java中，字符串对象在创建后是不可以改变的

    • String str = "abc"，则在程序中str不会变成"def"，也不会变成"ab"等

  • 所有的字符串直接存储在方法区内存的字符串常量池中

    • 字符串的使用非常频繁，为了提高执行效率，所以把字符串放到字符串常量池中
    • 字符串常量池不会被垃圾回收器回收
    • 方法区内存中还存储了字节码文件、代码片段、静态变量等等信息
    • 常量池也意味着字符串在创建后无法再修改
    • 特殊情况
    • 通过new方法创建的对象一定都在堆内存中，但是字符串都在方法区内存中

    • 字符串"abc"仍然存储在方法区的字符串常量池中，通过new方式在堆内存中创建了一个String对象，但是这个对象不存储字符串信息，而是存储着"abc"在方法区内存的地址，而栈内存中的str引用存储着堆内存中String对象的地址
    • 更进一步的

equals(Object anObject)方法

  • String类重写了equals()方法

Compares this string to the specified object. The result is true if and only if the argument is not null and is a String object that represents the same sequence of characters as this object（只有当两个字符串都包含相同的字符次序时才会返回真）.

String的构造方法

String(byte[] bytes)

Constructs a new String by decoding（解码） the specified array of bytes using the platform's default charset. The length of the new String is a function of the charset, and hence may not be equal to the length of the byte array.

  • 输出一个引用时会自动调用其toString()方法，默认输出对象的内存地址，String类已经重写了该方法

toString()

This object (which is already a string!) is itself returned.（返回字符串自己）

String(byte[] bytes, int offset, int length)

bytes - The bytes to be decoded into characters（待转换的字节数组）

offset - The index of the first byte to decode（待转换的起始位置）

length - The number of bytes to decode（待转换的长度）

  • bytes数组很长，我们只希望转换其中一部分为String时调用该构造方法

String(char[] value)

  • 与String(byte[] bytes)类似

String(char[] value, int offset, int count)

  • 与String(byte[] bytes, int offset, int length)类似

String(String original)

  • 通过已有的String来构造新的String
  • String str = new String("abc");

String中的常用方法

charAt(int index)

Returns the char value at the specified index（返回字符串中某个下标的字符值）. An index ranges from 0 to length() - 1. The first char value of the sequence is at index 0, the next at index 1, and so on, as for array indexing.
If the char value specified by the index is a surrogate, the surrogate value is returned.

compareTo(String anotherString)

Compares two strings lexicographically（按字典顺序比较两个字符串）. The comparison is based on the Unicode value of each character in the strings. The character sequence represented by this String object is compared lexicographically to the character sequence represented by the argument string. The result is a negative integer if this String object lexicographically precedes the argument string（str小于anotherString时返回负数）. The result is a positive integer if this String object lexicographically follows the argument string（str大于anotherString时返回正数）. The result is zero if the strings are equal; compareTo returns 0 exactly when the equals(Object) method would return true.（当两个字符串相等时返回0）

  • str1.compareTo(str2)
    • 等价于str1 - str2
    • 当str1大于str2时返回正数
    • ...小于...返回负数
    • ...等于...返回0

IndexOf(String str)

Returns the index within this string of the first occurrence of the specified substring, starting at the specified index.（返回字符串中某一子串第一次出现的下标）

public int lastIndexOf(String str)

Returns the index within this string of the last occurrence of the specified character.（返回字符串中某一子串最后一次出现的下标

public boolean contains(CharSequence s)

Returns true if and only if this string contains the specified sequence of char values.（当字符串包含某一部分字符串时返回真）

  • 字符串str包含字符串s等价于s出现在str中

public boolean endsWith(String suffix)

Tests if this string ends with the specified suffix.（判断某个字符串是否以某一个子串结尾）

  • 类似的也有startWith(String prefix)

public boolean equalsIgnoreCase(String anotherString)

Compares this String to another String, ignoring case considerations. （在忽略大小的区别下，比较两个字符串）Two strings are considered equal ignoring case if they are of the same length and corresponding characters in the two strings are equal ignoring case.

public boolean isEmpty()

Returns true if, and only if, length() is 0.（判断某个字符串长度是否为0）

  • 当一个字符串长度为0时，返回true
    • 判断数组长度时，是调用length属性
    • 判断字符串长度时，是调用length()方法
  • 长度为0并不等价于该字符串为null
    • 空引用调用方法时会出现空指针异常的错误

public String replace(CharSequence target, CharSequence replacement)

Replaces each substring of this string that matches the literal target sequence with the specified literal replacement sequence.（将字符串中每一个target子串，用replacement串替换） The replacement proceeds from the beginning of the string to the end, for example, replacing "aa" with "b" in the string "aaa" will result in "ba" rather than "ab".

public String[] split(String regex)

Splits this string around matches of the given regular expression（用给定的正则表达式拆分原字符串）

public String substring(int beginIndex, int endIndex)

Returns a string that is a substring of this string. The substring begins at the specified beginIndex and extends to the character at index endIndex - 1. （从字符串中截取从beginIndex到endIndex-1之间的子串）（左闭右开）Thus the length of the substring is endIndex-beginIndex.

public char[] toCharArray()

Converts this string to a new character array.（将字符串转换为字符数组）

public String toLowerCase()

Converts all of the characters in this String to lower case using the rules of the default locale. （将字符串中所有的字母都转化为小写）

public String toUpperCase()

Converts all of the characters in this String to upper case using the rules of the default locale. （将字符串中所有的字母都转化为大写）

public String trim()

Returns a string whose value is this string, with all leading and trailing space removed, where space is defined as any character whose codepoint is less than or equal to 'U+0020' (the space character).（去掉字符串中前面以及后面的空格，中间的空格不会去掉）

public static String valueOf(Object obj)

Returns the string representation of the Object argument.（将Object obj用字符串的形式表示）（少有的静态方法）

  • valueOf在底层调用的是toString()方法
  • println方法在底层调用的是valueOf

  • 所有能在控制台上显示的内容都是字符串

    • 将数据通过valueOf转换成字符串

StringBuffer

StringBuffer的原理

  • 由于字符串是不可变的，每一次拼接都会产生一个新的字符串，这样会占用大量的方法区内存，造成空间的浪费

StringBuffer的存储原理

  • public StringBuffer()

Constructs a string buffer with no characters in it and an initial capacity of 16 characters.

  • StringBuffer的底层是一个byte数组
  • 初始化容量是16

StringBuffer的扩容

  • public synchronized StringBuffer append(CharSequence s)

Appends the specified CharSequence to this sequence.

  • 通过append方法追加追加追加内容时，如果byte数组满了，就通过数组拷贝的方式进行自动扩容

  • String类型的底层存储也是通过byte数组，但是该数组由final修饰，无法修改！

    • 准确来说一个引用只能指向一块空间，如果扩容了一个新的空间，原来的引用无法指向它

  • 提高StringBuffer的性能：在创建时就给定一个初始化容量，最好减少底层数组的扩容次数

StringBuffer和StringBuilder的区别

  • StringBuffer中的方法都有synchronized关键字修饰，表示StringBuffer在多线程环境下运行是安全的
  • StringBuilder中的方法没有被synchronized修饰，表示在多线程运行情况下是不安全的
