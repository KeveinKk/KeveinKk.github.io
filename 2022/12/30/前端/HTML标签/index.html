排版标签

标题标签（Heading）

场景

  • 在新闻和文章的页面中，都离不开标题，用来突出显示文章主题

代码

  • h系列标签
  • HTML标题是通过<h1> - <h6> 标签来定义的
  • 1～6级标题，重要程度依次递减
    • 标题独占一行
    • 不同的等级文字加粗程度不同、大小不同

段落标签

场景

  • 在文章的页面中，用于分段显示内容

代码

  • HTML 段落是通过标签<p>来定义
  • 段落之间存在间隙
  • 一个段落独占一行

WebStorm代码自动换行

  • 在WebStorm中编辑代码时，如果一行代码太长可以选择自动换行功能
  • 点击代码行号的灰色部分，勾选Soft-Wrap即可

换行标签

场景

  • 让文字强制换行显示

代码

  • <br>

  • 单标签
  • 可以让文字强制换行

水平线标签

场景

  • 分割不同主题内容的水平线

代码

  • <hr>

文本格式化标签

场景

  • 需要让文字加粗、下划线、倾斜、删除线等效果
    • 在Typora中删除线的快捷键是control + shift +`

代码

标签 标签 说明
b strong 加粗
u ins 下划线
i em 倾斜
s del 删除线

  • 当希望强调某种效果时，使用单词标签，否则就使用字母标签即可

媒体标签

场景

  • 在网页中显示图片

标签的属性

  • 标签属性由两部分组成
    • 属性名
    • 属性值
  • 标签的属性卸载开始标签内部
  • 标签上可以同时存在多个属性
    • 多个属性之间用空格隔开
    • 多个属性之间没有顺序之分
  • 标签名与属性之间一定要用空格隔开

路径

  • 绝对路径
    • 目录下的绝对位置，可直接到达目标位置，通常从根目录开始的路径
    • 或者完整的网络地址
    • 不同的电脑下绝对路径会不相同，所以一般不使用绝对路径
  • 相对路径
    • 从当前文件开始出发寻找目标文件的过程
  • 相对路径的分类
    1. 同级目录
      • 当前文件和目标文件在同一个目录中
      • 相对路径直接为文件名或者./文件名
    2. 下级目录
      • 目标文件在下级目录中
      • 相对路径为目标文件所在的目录/文件名
    3. 上级目录
      • 目标文件在上级目录中
      • 相对路径为../目标文件所在的目录/文件名
      • 通过..来返回上一级目录

图片标签

  • <img src="" alt="">
  • 单标签
  • img标签需要展示对应的效果，需要借助标签属性进行设置
    • src=""以及alt=""都是img标签的属性

src属性

  • src是img标签的源属性，源属性的值是图像的URL地址
    • 即存储图像的地址
    • 绝对路径或者相对路径都可以

alt属性

  • alt属性用来为图像定义一串预备的可替换的文本
    • 在浏览器无法载入图像时，替换文本属性告诉用户失去的信息
    • 当图片加载失败时，才显示替换文本
    • 当图片加载成功时，不会显示替换文本

height和width属性

  • height属性用于设置图片的高度
    • 属性值默认单位为像素
  • width属性用于设置图片的宽度
    • 属性值默认单位为像素
    • 如果只设置了高度或者宽度中的一个，另一个会自动等比例缩放（图片不会变形）
    • 建议在使用图片时指定图片的高度或宽度，这样页面加载时就会保留指定的尺寸。如果没有指定图片的大小，加载页面时有可能会破坏HTML页面的整体布局。

title属性

  • title属性用于显示提示文本，当鼠标悬停时才显示的文本

音频标签

  • <audio src=""></audio>
  • audio标签在HTML4中是无效的
  • 只支持三种格式的音频文件
    • Mp3 、Waw、Ogg

controls属性

  • 显示播放的控件

autoplay属性

  • 自动播放
  • 部分浏览器不支持该功能

loop属性

  • 循环播放

视频标签

  • <video src=""></video>
  • 同样有controls autoplay loop三种属性
    • autoplay需要和muted（静音）一起使用才能实现自动播放

链接标签

作用

  • 点击之后，从一个页面跳转到另一个页面

  • 又称为a标签、超链接、锚链接

  • 双标签，内部可以包裹内容

href属性

  • href属性中添加跳转地址
    • 需要跳转到外部网页的话需要填入完整的URL
    • 需要跳转到内部网页的话填入路径
    • href="#"代表空链接

target属性

  • 属性值：目标网页的打开形式
属性值 效果
_self 默认值，在当前窗口中跳转（覆盖原网页）
_blank 在新窗口中跳转（保留原网页）

列表标签

应用场景

  • 场景
    • 在网页中按照行展示关联性的内容，如新闻列表、排行榜、账单等等
  • 特点
    • 按照行的方式，争气显示内容
  • 种类
    • 无序列表、有序列表、自定义列表

无序列表

作用

  • 在网页表示一组无顺序之分的列表

标签组成

标签名 说明
ul 表示无序列表的整体，用于包裹li标签
li 表示无序列表的每一项，用于包含
  • ul标签中只允许包含li标签
    • ul和li是嵌套关系
  • li标签中可以包含任意内容
  • 显示特点
    • 列表的每一行前默认显示圆点表示

代码

有序列表

作用

  • 在网页中表示一组有顺序之分的列表，如排行榜等

标签组成

标签名 说明
ol 表示有序列表的整体，用于包裹li标签
li 表示有序列表的每一项，用于包含每一行的内容
  • ol标签中只允许包含li标签
  • li标签中可以包含任意内容
  • 列表的每一项前默认显示序号标识

代码

自定义列表Description List

作用

  • 在网页的底部导航中通常会使用自定义列表来实现

标签组成

标签名 说明
dl 表示自定义列表整体，用于包裹dt和dd标签
dt 自定义列表的主题
dd 表示自定义列表针对主题的每一项内容
  • 自定义列表中每一项前会默认显示锁进效果
  • dl标签中只能包含dt和dl
  • dt和dl中可以包含任意内容

表格标签

表格的基本标签

标签名 说明
table 表格整体，用于包裹多个tr
tr 表格的每行，用于包裹td
td 表格单元格，用于包裹内容
th 表格每一列的主题
caption 表格的标题
  • table用于包含tr
  • 有几行就需要几对tr
  • tr可以包含th，也可以包含td
  • 一对td或者th只能包含一个单元格的内容
  • 表格的标题默认在表格整体顶部居中位置显示
  • caption标签书写在table标签内部

表格相关的属性

border

  • 为表格增加边框
  • 属性值为边框的宽度，单位是像素

width

  • 为表格的宽度
  • 属性值为边框的宽度，单位是像素

heigth

  • 为表格的高度
  • 属性值为边框的高度，单位是像素

表格的结构标签

  • 作用
    • 让表格的内容结构分组，突出表格的不同部分（头部、主体、底部），使语义更加清晰
标签名 名称
thead 表格头部
tbody 表格主体
tfoot 表格底部
  • 表格结构标签可以省略
  • 表格结构变迁内部用tr标签包裹内容

合并单元格

作用

  • 将水平或垂直多个单元格合并成一个单元格
  • 跨行合并：垂直合并成一个
  • 跨列合并：水平合并成一个

代码

  • 首先明确合并哪几个单元格
  • 通过左上原则，确定保留谁删除谁
    • 跨行合并：只保留最上面的
    • 跨列合并：只保留最左边的
  • 给保留的单元格设置：跨行合并（rowspan）或者跨列合并（colspan）
属性名 属性值 说明
rowspan 合并单元格的个数 跨行合并
colspan 合并单元格的个数 跨列合并
  • 只有同一个结构标签中的单元格才能合并，不能跨结构标签合并
  • 被合并的元素需要手动删除

表单标签

表单

  • 表单是一个包含表单元素的区域
  • 表单元素是允许用户在表单中输入内容，比如：文本域（textarea）、下拉列表（select）、单选框（radio-buttons）、复选框（checkbox） 等等

input系列标签

作用

  • 在网页中收集用户输入的内容

代码

  • 标签名：input
    • input标签可以通过type属性值的不同展示不同的效果
标签名 type属性 说明
input text 文本狂，用于输入单行文本
input password 密码框，用于输入密码
input radio 单选框，用于多选一
input checkbox 多选框，用于多选多
input file 文件选择，用于上传文件
input submit 提交按钮，用于提交
input reset 重置按钮，用于重置
input button 普通按钮，默认无功能，配合JS实现其他功能

placeholder属性

  • 占位符，用于提示用书输入内容的文本
  • 默认是浅色的字体
  • 注意区分placeholder和value属性之间的区别
    • placeholder是占位符，不是真正的值，只起到提示的作用
    • value是真正输入到input标签中的值

input Type text

  • 显示的内容就是输入的内容

input Type password

  • 输入的内容无法直接看到

input Tpye radio

  • 在多个单选框中只能选择一个
  • 默认情况下radio并没有实现单选功能
  • name属性
    • 具有部分组的作用
    • 有相同name属性值的单选框为一组，一组中只有一个单选框可以被选中
  • checked属性
    • 默认选中

input Type checkbox

  • 多选框中的内容可以勾选多个

input Type file

  • 点开选择文件即可上传文件
  • multiple属性
    • 允许一次性上传多个文件

按钮标签

type属性值

标签名 type值 说明
input/button submit 提交按钮，点击之后提交数据给后端服务器
input/button reset 重置按钮，点击之后恢复表单默认值
input/button button 普通按钮，默认无功能，配合JS添加功能

value属性

  • 用于修改按钮上所显示的文字
  • submit和reset按钮默认显示提交和重置

表单域标签

  • 使用form标签把表单标签包裹起来一起使用，用来实现按钮的功能

select下拉菜单标签

  • 在网页中提供多个选项的下拉菜单空间

  • 标签组成

    • select标签：下拉菜单的整体
    • option标签：下拉菜单的每一项
      • selected属性代表默认选中的选项
      • 默认情况下下拉菜单的第一个选项是默认选中的个选项
    • select标签用于包裹option标签

textarea文本域标签

  • 在网页中提供可输入多行文字的表单控件
  • 可以实现自动换行

常见属性

  • cols：规定了文本域可见的宽度
  • rows：规定了文本域可见的行数

label标签

  • 用于绑定内容与表单标签的关系
  • 以单选表单为例：
    • 在之前的单选表单中只有选中了前面的单选框才能勾选该选项
    • 将单选表单与单选项的内容绑定后，通过单选项的内容也能选择该项

使用方法1

  • 使用label标签将内容包裹起来
  • 在表单标签中添加id属性
  • 在label标签的for属性中设置对应的id属性值
    • 将label的for属性的值和表单标签中id属性的值统一
  • 区分name属性和id属性
    • name属性是用于将多个单选框设置成一个总体
      • 在这个总体里，只能有一个单选项被选中
      • 在这个总体里的每一个单选表单的name值都相同，才代表它们都在这一个总体里
    • id属性是用于将label标签和表单标签的内容联系在一起
      • 需要被捆绑在一起的表单标签的id属性和对应的label标签的for属性相同

使用方法2

  • 直接使用label标签将内容和表单标签包裹在一起
  • 将label标签中的for属性删除即可

语义化标签

无语义的布局标签div和span

  • div标签
    • 被该标签包裹的内容没有任何特殊效果，只是对其包裹的内容在排版上有特殊要求
    • 被div包裹的内容独享一行

  • span标签
    • 一行可以显示多个

有语义的布局标签

  • 适用于移动端的网页制作
有语义的布局标签
标签名 语义
header 网页头部
nav 网页导航
footer 网页底部
aside 网页侧边栏
section 网页区块
article 网页文章

字符实体

  • 能够通过字符实体在网页中显示特殊符号
  • 浏览器无法识别一些特殊符号（比如空格、大于号、小于号等等）
  • 常见字符实体
  • 区分大小写，同时以分号结束
  • 空格的实体名称&nbsp;
