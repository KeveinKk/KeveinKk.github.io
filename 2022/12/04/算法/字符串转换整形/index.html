字符串转换整形

题目

请你来实现一个myAtoi(string s)函数，使其能将字符串转换成一个 32 位有符号整数。

函数myAtoi(string s)的算法如下：

  1. 读入字符串并丢弃无用的前导空格
  2. 检查下一个字符（假设还未到字符末尾）为正还是负号，读取该字符（如果有）。 确定最终结果是负数还是正数。 如果两者都不存在，则假定结果为正。
  3. 读入下一个字符，直到到达下一个非数字字符或到达输入的结尾。字符串的其余部分将被忽略。
  4. 将前面步骤读入的这些数字转换为整数（即，”123” -> 123， “0032” -> 32）。如果没有读入数字，则整数为 0 。必要时更改符号（从步骤 2 开始）。
  5. 如果整数数超过 32 位有符号整数范围 [−231, 231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被固定为 −231 ，大于 231 − 1 的整数应该被固定为 231 − 1 。
  6. 返回整数作为最终结果。
  7. 注意：
    • 本题中的空白字符只包括空格字符 ‘ ‘
    • 除前导空格或数字后的其余字符串外，请勿忽略 任何其他字符

分析

这道题主要考察对输入种类的判断，以及边界情况（溢出）的处理。思路上没有什么难点，但是在细节上有很多需要注意的。

  • 对溢出的判断

    1. 方法一

      • 每一轮的结果计算公式如下：$res=res\times 10+temp$
      • 用一个变量pre记录上一轮的计算结果，res代表这一轮的计算结果
        • 如果发生溢出的话$pre\ne res {\div} 10$

    2. 方法二

      • int型发生溢出的话我们考虑使用long型来存储数据

      • 用一个long型的变量res来记录每一轮的计算结果

        • 如果是正数发生溢出
        • 如果是负数发生溢出

  • 将字符转化为int

    1. 通过Character类中的方法

      • public static int getNumericValue(char ch)

        Returns the int value that the specified Unicode character represents

    2. 通过字符之间的运算

  • 时间复杂度：遍历一遍字符串即可，$O(N)$

  • 空间复杂度：$O(1)$
