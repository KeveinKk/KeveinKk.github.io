访问控制权限

访问控制权限修饰符

  • 访问控制权限修饰符用来控制元素的访问范围
  • 修饰符包括：public, protected, 缺省 以及 private
  • 访问控制权限可以修饰 类 变量 方法
  • 修饰的范围：private < 缺省 < protected < public
  • 类只能用public和缺省修饰【内部类除外】

public

  • 表示公开的，在任何位置都可以访问

protected

  • 表示受保护的，在同一个包，以及子类中可以访问
  • 当某个数据只希望子类使用时，采用protected修饰

缺省

  • 仅仅是同一个包下可以访问

private

  • 私有的，只能在本类中访问
