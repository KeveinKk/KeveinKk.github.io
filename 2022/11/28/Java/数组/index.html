数组

  • Java中的数组是一个引用数据类型，其父类是Object
    • 数组存储于堆内存中
  • 数组实际上是一个容器，可以容纳多个元素
  • 数组中可以存储基本数据类型，也可以存储引用数据类型
    • 存储引用属于类型时，存储的是对象的引用
  • 在Java数组一旦创建，中长度不可变
  • 数组的分类
    • 一维数组：最常用
    • 二维数组
    • 多维数组：很少用
  • 所有的数组对象都有length属性，用来获取数组中元素的个数
  • Java要求数组中的元素类型统一
    • int型数组只能存储int型元素
  • 数组中元素的内存地址是连续的
  • 数组中首元素的内存地址作为整个数组对象的内存地址
    • 数组中元素的内存地址是连续的，知道第一个元素的地址就可以推算出其他元素的地址

数组的优缺点

  • 优点
    1. 检索效率高
      • 知道第一个元素的内存地址、知道每个元素占用的空间大小，可以通过下标直接定位出每个元素的内存地址
    2. 存储效率高
  • 缺点
    1. 增加/删除元素效率低
      • 对于数组中最后一个元素的增删是没有效率影响的
    2. 数组很难存储大量的数据
      • 无法在内存中开辟一大块连续的空间

初始化

  • 分为静态初始化和动态初始化

  • 静态初始化

    • 当创建数组时就确定数组中存储的元素，那么采用静态初始化

  • 动态初始化

    • 通过动态初始化的数组元素默认值为0
    • 预先为数组元素分配内存空间，在后期可以修改数组元素的值

数组的遍历

  • ArrayIndexOutOfBoundsException

    • 数组下标越界异常：当访问的元素的下标超出或者小于数组元素的范围时抛出的异常

  • For - Each循环

    • 可以在不适用下标的情况下遍历数组

数组作为方法的参数

  • 数组可以作为参数传递给方法

  • 直接传递静态数组的方法

main方法的形参列表

  • JVM负责调用main方法，main方法的形参列表是String[] args

  • JVM传递的实参并不是一个空引用，但并没有存储数据

  • String[] args是用来接受用户在控制台上输入的参数的

    • 用户通过java命令来输入参数
      • java + 类名 + 参数
    • JVM将用户输入的参数按照空格的方式进行分离，分离转化完成后自动放到String[] args中

引用类型的数组

  • 当数组中存储引用数据类型时，会涉及到多态的概念

数组的扩容

  • 扩容的过程
    • 当原数组元素满了的时候
    • 新建一个大容量的数组，将之前数组中的数据一个一个的拷贝到大数组中
  • 扩容的效率很低
    • 在设置数组时需要提前估计好数组的长度，避免发生拷贝

数组的拷贝

  • System.arraycopy(Object src,int srcPos,Object dest, int destPos, int length)

    Copies an array from the specified source array, beginning at the specified position, to the specified position of the destination array. A subsequence of array components are copied from the source array referenced by src （从src数组中复制）to the destination array referenced by dest（复制到dest数组中）. The number of components copied is equal to the length argument（拷贝元素的长度）. The components at positions srcPos through srcPos+length-1 in the source array are copied into positions destPosthrough destPos+length-1, respectively, of the destination array.

    • 用Object可以更好的兼容各种数组类型
    • 所有的数组都继承了Object类

  • 从src数组的srcPos的位置开始复制，拷贝到到dest数组从destPos位置开始的地方，一共复制length个元素，

  • 拷贝的内容不仅可以拷贝基本类型的数据，还可以拷贝引用类型的数据

    • 拷贝引用类型的数据时拷贝的是其地址
