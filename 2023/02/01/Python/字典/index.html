Python字典

字典

  • 字典时另一种可变容器模型，且可以存储任意类型对象
  • 字典以键值对的形式存储数据
    • 键值对用冒号分隔
    • 每对之间用逗号分隔
    • 整个字典包括在花括号中
  • 字典的命名不要使用dict
    • dict是Python的关键字和内置函数
  • 键的值必须是唯一的，但值可以有重复
  • 值可以取任何数据类型，但键的值必须是不可变的，例如字符串、数字等等

字典的特性

  • 字典的值可以时任何Python对象，既可以时标准对象，也可以是用户定义的
  • 但是键不可以，只能是不能改变的数据类型
  • 同一个键不能出现两次
    • 创建时如果同一个键被赋值两次，后一个的值会覆盖前一个的值
  • 键必须是不可以改变的，可以使用数字、字符串或者元组，但是不能使用列表

创建空字典

  • 直接使用大括号就可以创建空字典
  • 使用函数dict()创建字典

访问字典的值

  • 将相应的键放入到方括号中，就可以实现对对应值的访问
    • 注意如果键的属性是字符串，那方括号中也必须有引号
  • 不能访问不存在的键值对，会报错KeyError

修改字典

  • 增加键值对
  • 修改原有的键值对
    • 只能修改键值对中的值，不能修改键

删除字典

  • 通过del命令删除字典
  • 通过del删除字典后，整个字典会消失，而不是变成空
    • 再次访问该字典时会报错：name 'dic' is not defined

字典内置的函数以及方法

Python字典包含了以下内置函数

  • len(dict)
    • 计算字典元素个数，即键的总数
  • str(dict)
    • 输出字典，用可以打印的字符串表示
  • type(variable)
    • 返回输入的变量类型

Python字典包含了以下内置方法

  • dict.clear()
    • 删除字典内所有的元素
    • 删除后字典变成空字典，但是还能继续访问该空字典
  • dict.copy()
    • 返回一个字典的浅复制
  • dict.fromkeys(seq[, value)
    • 创建一个新的字典，以序列seq中元素做字典的键，val为字典所有键对应的初始值
    • val如果省略的话，所有的键对应的值为None
  • dict.get(key, default=None)
    • 返回指定的键的值，如果键不在字典中返回default中设置的默认值
    • 等价于dict.setdefault(key, default=None)
  • key in dict
    • 如果键在字典中返回true，否则返回false
  • dict.items()
    • 以列表的形式返回一个视图对象
    • 将字典转换成一个列表，并返回该列表
  • dict.keys()
    • 返回键的视图对象
    • 与上一个方法类似，但这里只转化键，不是将整个键值对都转化
  • dict.value()
    • 返回值的视图对象
    • 注意：视图对象是动态的，首字典变化的影响
      • 如果修改了字典中的某个键值对，则keys和values的视图对象也会有变化
  • dict.update(dic2)
    • 将字典dic2中的键值对更新到字典dict的后面
  • pop(key[,default])
    • 删除字典key所对应的值，并返回被删除的值
    • 当键不存在时，返回默认设置的值
  • popitem()
    • 返回并删除字典中的最后一对键值对
    • 返回值类型是元组，即以元组的类型返回最后一堆键值对
