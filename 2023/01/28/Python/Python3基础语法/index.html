Python3基础语法

发表于 2023-01-28 更新于 2023-01-29 分类于 Python
Python学习-Python3基础语法

Python3基础语法

标识符

  • 第一个字符必须是字母表中字幕或下划线_
  • 标识符的其他部分由字母、数字和下划线组成
  • 标识符对大小写敏感
  • 不能将关键字作为标识符的名称

注释

  • 单行注释以#开头
  • 多行注释可以使用'''或者"""

行与缩进

  • Python使用缩进来代表代码块，不需要使用大括号
  • 缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的缩进空格数

多行语句

  • Python通常是一行写完一条语句，但如果语句很长，可以使用\来实现多行语句
  • 在括号中的多行语句不需要使用\

数字类型

  • Python中数字有四种类型，整数、布尔型、浮点数和复数

    1. int：只有一种整数类型，表示为长整形
    2. bool：只有首字母大写，如True，False
    3. float：同样也只有一种浮点数类型
    4. complex：复数

  • 数字类型都不可变的

判断数据类型

  • 使用isinstance()来判断一个变量的类型
  • 使用type()来判断一个变量的类型
  • isinstance()和type()的区别
    • type()不会认为子类是一种父类类型
    • isinstance()会认为子类是一种父类类型。

字符串

  • Python中单引号和双引号的使用完全相同
  • 三引号可以指定一个多行字符串

转义符

  • 转义符号\
  • 反转义符号r
    • 让转义符号不发生转义

字符串的操作

  • 字符串可以用+运算符连接在一起，用*运算符重复
  • Python中的字符串有两种索引方式
    • 从左往右以 0 开始：0 1 2 …
    • 从右往左以 -1 开始：-1 -2 -3 …
      • -1代表的是倒数第二个字符
      • 倒数第一个字符默认不写
  • Python 中的字符串不能改变
    • 修改字符串会报错
    • string[0] = 'm'
  • Python 没有单独的字符类型，一个字符就是长度为 1 的字符串
  • 字符串的截取的语法格式如下：变量[头下标:尾下标:步长]
    • 其中步长默认为1时可以省略
    • 尾下标默认是最后一个字符时可以省略

数据类型转换

  • 数据类型转换分为两种
    • 隐式类型转换：自动完成
    • 显式类型转换：需要使用类型函数来转换

隐式类型转换

  • 在计算时，Python 会将较小的数据类型转换为较大的数据类型，以避免数据丢失

显示类型转换

  • int()
    • 强制转换为整型
    • 直接去掉小数部分
  • float()
    • 强制转换为小数
  • str()
    • 强制转换为字符串类型

空行

  • 函数之间或者类的方法之间用空行分隔，标识一段新的代码的开始
  • 类和函数入口之间也用一行空行分隔，突出函数入口的开始
  • 空行与缩进不同
    • 空行并不是Python语法的一部分，不书写空行解释器也不会报错

同一行显示多条语句

  • Python可以在同一行中使用多条语句，语句之间使用分号 ;分割

代码组

  • 缩进相同的一组语句构成一个代码块，或者代码组
  • 像if、while、def和class这样的复合语句，首行以关键字开始，以冒号( : )结束，该行之后的一行或多行代码构成代码。
    • 我们将首行及后面的代码组称为一个子句(clause)

import

  • 在Python用import或者from...import来导入相应的模块
  • 将整个模块(somemodule)导入，格式为： import somemodule
  • 从某个模块中导入某个函数,格式为： from somemodule import somefunction
  • 从某个模块中导入多个函数,格式为： from somemodule import firstfunc, secondfunc, thirdfunc
  • 将某个模块中的全部函数导入，格式为： from somemodule import *

输入函数input

  • 接受用户的输入
  • input(str)
    • 参数是一个字符串，代表提示信息
    • 函数的返回值也是一个str类型

运算符

算数运算符

运算符 描述
+ 两个对象相加
- 得到负数或者是一个数减去另一个数
* 两个数相乘或者是返回一个被重复若干次的字符串
/ 除
% 取模运算
** 幂运算
// 向下取整

位运算

运算符 描述
& 按位与
｜ 按位或
^ 按位异或
~ 按位取反
<< 左移
>> 右移

逻辑运算符

  • 以下例子中a为10，b为20
  • 在Python中所有的对象都有一个布尔值
    • 通过内置函数bool()获得对象的布尔值
  • 布尔值为False的对象
    • 数值0
    • None
    • 空字符串、空列表、空元组、空字典、空集合
运算符 逻辑表达式 描述 实例
and x and y 布尔与：如果x为假，返回x的值，否则返回y的值 a and b = 20
or x or y 布尔或：如果x为真，则返回x的值，否则返回y的值 a or b = 10
not not x 布尔非：如果x为真，返回假，否则返回真 False

成员运算符

  • 用于测试实例中是否包含了一系列的成员
运算符 描述
in 如果在指定的序列中找到值，返回True，否则返回False
not in 如果在指定的序列中没有找到值返回 True，否则返回 False

身份运算符

用于比较两个对象的存储单元

运算符 描述 实例
is 判断两个标识符是否引用自一个对象 x is y，如果引用的是同一个对象则返回True，否则返回False
is not 判断两个标识符是否引用自一个对象 相反

条件控制

if语句

  • Python中用elif代替了else if
  • if语句的关键字为if elif else
  • 每个条件后都要使用冒号，表示接下来是满足条件后要执行的代码块
  • 使用缩进来划分语句块
  • 在Python中没有switch case语句
  • 允许if的嵌套使用

match case

  • 与switch case语句类似，match后的对象会依次与case后的内容进行匹配，如果匹配成功，则执行匹配到的表达式，否则直接跳过
  • _类似default

循环语句

while循环

  • 同样需要注意冒号和缩进
  • 在Python中没有do…while循环
  • 无限循环
    • 通过设置条件表达式永远为真来实现无限循环
  • while循环使用else语句
    • 如果while后面的条件语句为false时，则执行else语句

for循环

  • for循环可以遍历任何可迭代的对象
  • for…else用于在循环结束后执行一段代码
    • 当循环执行完毕，会执行else中的代码
    • 循环过程中遇到了break语句，则会中断循环，不会再执行else中的语句

range()函数

  • range()函数返回一个可迭代的对象，而非列表类型

函数语法

  • start：计数从start开始。默认是从0开始，此时可以省略start
    • range(5)等价于range(0,5)
  • stop：计数到stop结束，但不包括stop
  • step：步长，默认是1，此时可以省略step
    • range(5)等价于range(0,5,1)
    • 步长可以是负数

配合for循环使用

  • 遍历数字序列
  • 结合range()和len()遍历序列

pass语句

  • pass是空语句，是为了保持程序结构的完整性
  • pass 不做任何事情，一般用做占位语句
  • 一般用于搭建语法结构的时候，在还没有想好具体代码怎么写的时候先用pass语句占位
