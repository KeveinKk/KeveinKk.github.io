JavaScript特殊的对象

JavaScript特殊的对象

Date

Date()

  • 通过Date()方法获得当前的日期

getFullYear()

  • 获得当前的年份
  • getMonth() getDate()等等

getTime()

  • 获得从1970年1月1日至今的毫秒数

setFullYear()

  • 设置具体时间

JSON

  • JSON 英文全称 JavaScript Object Notation
  • JSON是用于存储和传输数据的格式
  • JSON通常用于服务端向网页传递数据
  • 任何JS支持的类型都可以用JSON来表示

语法规则

  • 数据用键值对的形式存储
  • 数据由逗号分隔
  • 大括号保存对象
  • 中括号保存数组

JSON数据

  • 数据是键值对的形式存储
  • 用双引号包裹键值对
  • "键":"值"

JSON对象

  • JSON对象保存在大括号中
  • 一个对象中可以保存多个数据

JSON数组

  • JSON 数组保存在中括号内

转换函数

JSON.parse()

  • 用于将一个 JSON 字符串转换为 JavaScript 对象

JSON.stringify()

  • 用于将 JavaScript 值转换为 JSON 字符串
