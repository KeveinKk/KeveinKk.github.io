CSS

基础认识CSS

  • 层叠样式表Cascading style sheets
  • 作用：给页面中的HTML标签设置样式
  • 样式
    • 样式定义如何显示一个HTML元素
    • 样式通常存储在样式表中
    • 外部样式表可以极大的提升工作效率
    • 外部样式表通常存储在CSS文件中
    • 多个样式定义可以层叠为一个

语法规则

CSS的基本组成

  • CSS规则主要由两部分组成：选择器，以及一条或者多条的声明
    • 选择器：用于选择需要改变样式的元素
    • 每条声明由一个属性和一个值组成
    • 属性是需要设置的样式属性，每个属性有一个值，属性和值用冒号分开
    • 每条属性之间用分号隔开
      • 在HTML里一个标签的不同属性用空格隔开即可

CSS的注释

  • 使用/* */作为注释

CSS的引入方式

  • CSS的引入方式一共分为3种，包括内嵌式、外联式、行内式

内嵌式

  • CSS写在style标签中，将CSS内嵌在HTML中
  • style标签虽然可以写在页面的任意位置，但是通常约定写在head标签中

外联式

  • CSS写在一个单独的.css文件中
  • 需要通过link标签在网页种引入
    • link标签也一般写在head标签中

  • rel属性指的是HTML文件和需要链接的文件的关系

    • 这里引入的CSS文件是HTML文件的央视文件

  • herf属性指的是引入文件的链接

  • 在.css文件中直接写CSS语句即可

行内式

  • CSS写在标签的style属性中

三种方式的比较

引入方式 书写位置 作用范围 使用场景
内嵌式 HTML文件的style标签中 当前页面 小案例中
外联式 单独创建一个.css文件，用link标签进行链接 可以作用于所有与之链接的HTMl文件 项目中
内行式 写在标签的style属性中 当前标签 配合js进行使用

基础选择器

  • 选择器用于选择需要改变样式的元素

标签选择器

结构

  • 标签名 {属性名：属性值;}

作用

  • 通过标签名找到页面中所有的这列标签，设置格式
  • 标签选择器选择的是一类标签，而不是单独的一个
  • 标签选择器无论嵌套的关系有多深，都能找到对应的标签

类选择器

结构

  • .类名{属性名：属性值;}
  • 类名需要自己定义，类名前有一个点

作用

  • 所有标签上都有class属性，class属性的属性值称为类名
  • 类名可以由数字、字母、下划线、中划线组成，但不能以数字或者中划线开头
  • 一个标签可以同时有多个类名，类名之间通过空格隔开
    • 多个类名都写在class属性中
  • 类名可以重复，一个类选择器可以同时选中多个标签

id选择器

结构

  • #id属性值{属性名：属性值;}
  • id值需要自己定义，id值之前需要一个#

作用

  • 通过id值，找到页面中带有这个id值的标签，设置样式
  • 所有的标签都有id属性
  • id属性在一个页面中是唯一的，不可重复
  • 一个标签上只能有一个id属性
    • 但是一个标签上可以有多个类
  • 一个id选择器一次只能选择一个标签

通配符选择器

结构

  • *{属性名：属性值;}

作用

  • 找到页面中所有的标签，设置样式
  • 只有在极少数情况下才会使用
    • 可以用于清楚所有标签默认的margin和padding（间距）
