优先级

特性

  • 不同选择器具有不同的优先级
  • 优先级较高的样式选择器会覆盖优先级较低的样式选择器

优先级公式

继承 < 通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important

  • 选择器选择的越精准，优先级越高
  • 选择器选择的范围越广，优先级越低

注意

  • !important写在属性值的后面，分号的前面
  • !important不能提升继承的优先级，继承的优先级最低
  • 并不建议使用!important

权重叠加计算

  • 如果是符合选择器，需要通过权重叠加计算的方法，判断哪个选择器优先级最高

规则

  • 复合选择器（行内样式的个数，id选择器的个数，类选择器的个数，标签选择器的个数）
    • 先计算权重高的选择器个数
    • 没有进位
  • 先比较第一级（行内样式的个数），有大小之分则忽略后面的
    • 否则继续比较第二级
  • 如果最终所有数字都相同，则优先级相同，适用CSS的层叠性
    • 哪个效果写在后面，哪个效果生效
  • !important权重最高，可以忽略一切规则
