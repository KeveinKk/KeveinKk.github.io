Javascript入门

简介

  • HTML定义了网页的内容
  • CSS描述了网页的样式
  • JavaScript控制了网页的行为

脚本语言

  • JavaScript是一种轻量级的脚本编程语言
  • JavaScript是可以插入HTML页面的编程代码
  • 脚本语言（英语：Scripting language）
    • 是为了缩短传统的编写、编译、链接、运行过程而创建的计算机编程语言。早期的脚本语言经常被称为批处理语言或作业控制语言。
    • 一个脚本通常是解释运行而非编译。脚本语言通常都有简单、易学、易用的特性，目的就是希望能让程序员快速完成程序的编写工作

HelloWorld

Javascript文件

  • HTML 中的 Javascript 脚本代码必须位于 <script> 与 </script> 标签之间。
  • Javascript 脚本代码可被放置在 HTML 页面的 <body> 和 <head> 部分中

<script>标签

  • 如需在 HTML 页面中插入 JavaScript，需要使用 <script> 标签
  • <script> 标签是成对出现的
  • <script> 和 </script> 会告诉Javascript在何处开始以及在何处结束

Javascript文件的引入

  • 内部标签
    • Javascript文件就在HTML文件中
    • 一般就卸载head标签中或者body标签末尾
  • 外部引入
    • 单独编写一个Javascript文件，然后在HTML文件的头部引入该JS文件
    • CSS文件借助link标签引入，Javascript文件借助script标签引入
  • 注意：
    • 即使在引入过程中<script>标签同样是成对出现的
    • <link>标签不需要
    • 在外部的Javascript中不再需要用<script>标签包裹，直接编写Javascript代码即可

语法入门

变量

变量名

  • 变量必须以字母开头，甚至可以以$和_开头
  • 变量名对大小写敏感

数据类型

  • Javascript不存在各种数据类型，所有的数据类型的都用var表示

变量的声明和赋值

  • 使用var关键字来对变量进行声明
  • 变量在声明后是空的，需要对其进行赋值
  • 当然，和Java类似的，我们可以在一条语句中声明多个变量

条件语句

  • 和Java一样

调试

  • 在浏览器中的检查界面，在source界面可以添加断点进行调试
  • console.log()方法类似Java中的sout

数据类型

数字

  • JS不区分小数和整数
  • 同样可以使用科学技术法

字符串、布尔值、逻辑运算

  • 和Java一样

比较运算

  • 等于==
    • 值相等，即使类型不同也会返回true
  • 绝对等于
    • 值相等，而且类型也需要相同，才返回true

  • 类似的还有不等于!=和绝对不等于===

    • 建议都使用绝对等于

  • NaN(Not a Number)

    • 严格区分大小写
    • NaN不等于任何数字，并且也不等于自己

严格检查模式

  • 'use strict'代表严格检查模式
  • 预防JS语法的随意性所带来的各种问题
  • 'use strict'必须写在JS代码的第一行
