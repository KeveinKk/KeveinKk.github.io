用数组模拟栈

题目

  • 使用一个一维数组模拟栈的数据结构
  • 这个栈可以存储Java中任何的引用类型的数据
  • 在栈中提供push方法模拟压栈（栈满有提示）
  • 在栈中提供pop方法模拟弹栈（栈空有提示）

总结

  • Object类型的数组可以存储任何类型的引用类型数据
    • Object[] arr = {new Animal(), new Uer(), "abc"}
    • 这些元素之间
      • 可以有关系：所有的元素都是同一个类，或者它们之间有继承关系
      • 也可以没有关系
  • 调用一个对象时，默认调用的是该对象所在类的toString()方法
  • 当方法必须有返回值时，又需要返回一个空或者错误信息
    • 返回null
  • 每写一个类都要重载其toString()和equals()方法
